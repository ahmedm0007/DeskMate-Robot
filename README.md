# 🤖 DeskMate — Intelligent Study Assistant Robot

> A physical desktop robot that monitors student concentration in real time, detects distraction and fatigue, and provides personalized academic assistance through voice interaction and AI-powered document Q&A.

---

## 📌 Overview

DeskMate is an end-of-year engineering project built at **INSAT** (National Institute of Applied Science and Technology, Tunis). It addresses a concrete problem: study sessions are rarely as effective as they should be. Students get distracted, lose focus, and have no tool that actively intervenes.

DeskMate is a **physical companion robot** — present on your desk like a lamp — that observes, analyzes, and acts. Unlike passive apps, it adapts its behavior in real time to the student's actual state.

---

## ✨ Features

### 👁️ Behavioral Monitoring (Vision Pipeline)
- **Real-time face detection** and multi-user recognition (MediaPipe + ArcFace/DeepFace)
- **Fatigue detection** via Eye Aspect Ratio (EAR) and Mouth Aspect Ratio (MAR)
- **Distraction detection**: gaze direction (head pose estimation) + smartphone detection (YOLOv8)
- **Absence detection**: alerts when the student leaves the camera field
- Runs at **~30 FPS** on Raspberry Pi 4, well within the 500ms real-time constraint

### 🗣️ Voice Interaction (Vocal Pipeline)
- **Wake word detection** (local, via Vosk) → **Speech-to-Text** (Google Web Speech API)
- **LLM-powered intent classification** and conversational responses (Qwen2.5-7B-Instruct via Hugging Face API)
- **Text-to-Speech** output (edge_tts + pygame) with ~0.8s generation time
- End-to-end latency: **2–4 seconds** for a complete voice exchange

### 📚 Academic AI Assistant (RAG System)
- Upload your own **PDF course documents**
- Hybrid search: **semantic (ChromaDB)** + **keyword (BM25)** retrieval
- Answers generated by **Gemini 2.5 Flash**, grounded exclusively in your documents
- Evaluation results: **85% answer correctness**, 87.5% hit rate on test corpus

### 📅 Adaptive Study Planner
- Input your subjects, mastery level, exam dates, and available hours
- Generates a **personalized weekly revision schedule** using a priority scoring model
- Urgency factor recalculated daily as exams approach
- Subjects automatically removed after their exam date

### 📊 Session Management & Dashboard
- Supports **Pomodoro**, **Deep Work**, and **52/17** productivity techniques
- Voice-controlled session start/pause/stop
- OLED display shows current state (focused / distracted / break / session end)
- **React/Next.js web dashboard** (localhost): session history, productivity graphs, planning, PDF upload

### 🚗 Autonomous Mobility
- **Differential drive** (2× GA12-N20 motors) controlled by ESP32
- **ToF sensor (VL53L0X)** for desk-edge detection and collision avoidance
- Autonomous repositioning to keep the student in camera frame

## 🔧 Hardware

| Component | Role |
|---|---|
| Raspberry Pi 4 (4GB) | Main brain — runs all AI/software modules |
| ESP32 | Spinal cord — real-time actuator control |
| Pi Camera Module V2 (8MP) | Vision pipeline input |
| INMP441 Microphone (I2S) | Voice capture |
| MAX98357A Amplifier + 8Ω Speaker | TTS audio output |
| OLED Display SSD1306 (0.96") | Real-time status display |
| ToF Sensor VL53L0X | Desk-edge detection |
| 2× GA12-N20 DC Motors + TB6612FNG | Differential drive |
| 2× 18650 Li-ion (3000 mAh) + XL4015 Buck | Power supply (~2.5–3.5h autonomy) |
| 3D-printed PLA chassis | 3-level structure, custom-designed |
